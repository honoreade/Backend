Nodejs CRUD Examples

This demo shows two different approaches to implementing CRUD (Create, Read, Update, Delete) operations in Nodejs:

Traditional Approach

This implementation uses the traditional Nodejs approach with page reloads. Each action (create, read, update, delete) causes the page to reload.

Technologies: Nodejs, HTML, CSS

View Demo

Modern Approach

This implementation uses JavaScript and AJAX to perform CRUD operations without page reloads, providing a smoother user experience.

Technologies: Nodejs, JavaScript, Fetch API, HTML, CSS

View Demo

Technical Details

Both implementations use the same backend API and SQLite database. The difference is in how the frontend interacts with the backend:

  • Traditional: Form submissions and page reloads
  • Modern: JavaScript fetch API and DOM manipulation

The API endpoint (api-users.php) handles all CRUD operations and returns JSON responses.
